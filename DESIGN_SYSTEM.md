# Design System

Figma のデザインデータから抽出した、プロジェクト共通のデザインシステムです。
ダークテーマ・ライトテーマの両方に対応しています。

---

| `gray-100` | `#F2F2F2` | `#1A1A1A` | サブテキスト（ダーク）/ メイン文字（ライト） |

| `gray-300` | `#CCCCCC` | `#444444` | カード説明文 |

## 5. エフェクト（効果）

### 5-1. デザイン方針（Apple HIG 準拠）

要素の区切りには**枠線（border）ではなく、背景色の差と影**を使います。

| やること | やらないこと |
|---|---|
| 背景色の差で親子関係を表現 | 全ての要素に枠線をつける |
| 控えめな影で浮き上がりを表現 | 強い影（shadow-md以上）を多用 |
| ホバーで背景色が変化＋わずかに浮く | ホバーで枠線の色だけ変える |
| すりガラス効果で奥行きを出す | 不透明なヘッダー・タブバー |

### 5-2. ぼかし（Blur）

| トークン名 | 値 | 使いどころ |
|---|---|---|
| `blur-heavy` | `blur(30px)` | タブバーの背景（すりガラス風） |
| `blur-light` | `blur(4px)` | ステータスバー、ナビバーの背景 |

### 5-3. 影（Shadow）

枠線の代わりに影でカードや要素の階層を表現します。

| トークン名 | 使いどころ |
|---|---|
| `--shadow-card` | カード、入力フィールド、セクション |
| `--shadow-card-hover` | ホバー時の浮き上がり |
| `--shadow-float` | モーダル、ドロップダウンなど浮いた要素 |
| `--shadow-separator` | リスト項目の極薄の区切り線 |

### 5-4. ホバー・インタラクション

| 要素 | ホバー時の変化 |
|---|---|
| カード | 影が強まる＋`translateY(-1px)`で浮く |
| リスト行 | 背景色が `--color-bg-hover` に変化 |
| ボタン（Primary） | 色が `--color-primary-hover` に変化 |
| ボタン（Secondary） | 背景色が `--color-bg-hover` に変化 |
| タグ・ピル | 背景色がわずかに変化 |

## 7. レイアウトパターン

### 7-1. ページ構造

すべての画面は以下の基本構造に従います：

```
┌─────────────────────────┐
│     StatusBar (44px)     │  ← 時刻・電波・バッテリー
├─────────────────────────┤
│                         │
│     メインコンテンツ       │  ← スクロール可能エリア
│     （左右20pxマージン）   │
│                         │
├─────────────────────────┤
│   TabNavigation (80px)   │  ← 固定タブバー（すりガラス）
├─────────────────────────┤
│  HomeIndicator (21px)    │
└─────────────────────────┘
```

### 7-2. カルーセル（横スクロール）

```
← 20px →┌────┐ 8~15px ┌────┐ 8~15px ┌────┐
         │    │        │    │        │    │  ← 画面外へ続く
         └────┘        └────┘        └────┘
```

- 左側のみ20pxパディング
- アイテム間の隙間: 8〜15px
- 右端は画面外にはみ出して「続きがある」ことを示す

### 7-3. タブナビゲーション

```
  Today    Games    Apps    Arcade   Search
   🔵      ⚪       ⚪      ⚪       ⚪
```

- 5つのタブ: Today / Games / Apps / Arcade / Search
- アクティブ: Blue（`#147EDF`）
- 非アクティブ: Gray-500（`#989898`）
- 背景: すりガラス効果（`blur(30px)`）

## 8. テーマ切り替えの仕組み

CSSでは `data-theme` 属性を使ってテーマを切り替えます：

```html
<!-- ダークテーマ -->
<html data-theme="dark">

<!-- ライトテーマ -->
<html data-theme="light">
```

ユーザーのOS設定に合わせて自動切り替えも可能です。
詳細は `globals.css` のトークン定義を参照してください。
